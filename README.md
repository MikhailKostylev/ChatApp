# ChatApp
Simple real time chat app with SwiftUI and Firebase 🌐💌 

- 🔓 Firebase Authentication
- 📚 Firebase Cloud Firestore
- 🔎 Search for Users
- 👥 Real Time Conversations
- 🌚 Dark Mode Support
